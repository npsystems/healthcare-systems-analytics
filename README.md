Overview

This project was completed as part of an onboarding exercise during the beginning of my internship. 
It serves as an introduction to working with data, focusing on data cleaning, analysis, and report building. 
The exercise demonstrates my ability to take raw sales data and transform it into meaningful insights using Excel and Power BI.

Files & Viewing Order

To best understand the workflow and insights, please review the files in the following order:
  1. Sales Data & Excel Dashboard Workbook
      Contains the original dataset and the Excel-based dashboard
      Demonstrates data cleaning, transformation, and initial analysis
      Includes calculated fields and early KPI development
  2. Power BI Report File
      Interactive dashboard built from the cleaned dataset
      Highlights key metrics and visual insights
      Represents the final stage of report building
  3. Overview Header & Summary Screenshot
      Displays the report title and introductory summary
      Provides a high-level explanation of the analysis
      Data Tables & KPIs Screenshot
      Shows key performance indicators and supporting tables
      Summarizes the most important numerical insights
  4. Main Visualization Screenshot
      Central visual analysis from the Power BI dashboard
      Highlights trends, comparisons, and patterns in the data
  5. Context Analysis Screenshot
      Written interpretation of the data
      Explains what the results mean in a business context
  6. Future Considerations Screenshot
      Suggestions for further analysis or improvements
      Identifies potential next steps and opportunities

Tools & Skills Demonstrated
  1. Microsoft Excel
      Data cleaning and preparation
      Formula usage and KPI creation
      Initial dashboard development
  2. Power BI
      Data visualization
      Dashboard design
      Insight communication

File Information

Due to GitHub file size limitations, the full Excel dashboard is provided as a compressed .zip file.
Unzip the file to access and interact with the dashboard.

Notes

This was a guided onboarding exercise rather than a standalone business project.
Some calculations (e.g., LOIs) were used in the Excel dashboard but are not included in the Power BI report.
Due to layout constraints, certain sections (context and future considerations) are presented as separate screenshots.

How to Use
  1. Download the .zip file
  2. Extract the contents
  3. Open the Excel workbook
  4. Navigate to the dashboard sheet
